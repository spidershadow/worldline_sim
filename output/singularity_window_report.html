Limitations and Considerations

This analysis is subject to several important limitations:

  • GCP Data Simulation:
    • Current analysis uses simulated GCP data based on historical milestones
    • Only 2000-2015 values are based on actual GCP milestones
    • Post-2015 values are projections and may not reflect actual measurements
    • Interpolation between milestone points assumes linear progression
  • Data Quality: UAP sighting data varies in reliability and verification standards over time.
  • Selection Bias: GCP node locations are not uniformly distributed globally.
  • Correlation vs. Causation: Statistical correlation between GCP and UAP patterns does not necessarily imply causation.
  • Model Assumptions: The t* estimation model assumes temporal symmetry in anomaly patterns.
  • Alternative Explanations: Other factors not included in this analysis may better explain observed patterns.

UAP Data Limitations
